The code is a 2D dynamical fluid flow code using dry compressible non
hydrostatic equations modified from Matthew R. Norman original code.
The current implementation has been separated in multiple modules.

The original code is licensed with the accluded LICENSE.

## Requirement

To compile the code, a Fortran compiler is required, (the code has been tested
using gfortran version 13.3.0) and the netcdf-fortran library
[https://docs.unidata.ucar.edu/netcdf-fortran/current/]

To compile, GNU make is required. Just type in make.
